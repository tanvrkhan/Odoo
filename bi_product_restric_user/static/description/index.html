User Product & Category Restriction Odoo Apps

Product Restriction on Users Odoo app is designed to help businesses to streamline their sales processes by providing greater control over product visibility by restricting access to specific products, or product categories for particular users. Businesses can control who has access to different products and product categories by assigning products or product categories to specific users. With this app, businesses can limit access to sensitive products and prevent unauthorized sales.

Features

Restrict Access to Specific Products for Particular Users

Restrict access to specific products by assigning products to specific users.

Restrict Access to Specific Product Category for Particular Users

Restrict access to specific product categories by assigning product categories to specific users.

Configuration for Product Restriction on Users

Under Setting > User & Companies> Users menu enable the "Product Restriction on Users" user access.

Allow Products to User

Select the "Product" as "Restriction On" option and select all products that the user wants to allow access for a specific user.

Access Only Allowed Products

User can see and access only allowed products in any of the models.

Allow Product Category to User

Select the "Category" as "Restriction On" option and select all product categories that the user wants to allow access for specific user.

Access Only Allowed Product Category

User can see and access only allowed product categories in any of the model.
